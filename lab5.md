# Диаграмма классов


![lab5](https://github.com/YaYalii/rtipo/assets/131250193/c2ef901e-47e0-40dd-8e5d-a83e09c610db)


Класс Game содержит:
* Список игроков
* Список кубиков
* Раунд

Класс Game реализует методы:
* Сыграть раунд
* Получить значение броска кубиков
* Завершение игры
* Добавление игрока
* Оплата и подсчет выигрыша

Класс Dice содержит:
* Символы разметки кубика

Класс Dice реализует методы:
* Бросок кубика

Класс Bet Содержит:
* Значение ставки
* Символ на который делается ставка
